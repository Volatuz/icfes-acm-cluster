# ICFES Saber 11 – Tolima (2024)

Autor: German Pava  

Análisis exploratorio y multivariado de los resultados del ICFES Saber 11 (Tolima, 2024),
incluyendo descriptivos, pruebas no parametricas, análisis de correspondencias múltiples (MCA),
clustering y reglas de asociación.

---

## Sobre los scripts

Cada script **es independiente**.

- Todos cargan una **base de datos previamente limpia en 0_clean.R**
- No existe dependencia secuencial entre archivos
- Pueden ejecutarse de forma aislada

---
